# 部署說明

## 方法一：Cloudflare Pages（免費、快速）

1. 前往 https://pages.cloudflare.com
2. 登入 Cloudflare 帳戶
3. 點「Create a project」
4. 選擇「Direct upload」
5. 將 `out` 資料夾內容拖進去
6. 會得到一個 *.pages.dev 的網址

## 方法二：Vercel

1. 前往 https://vercel.com
2. 將專案上傳到 GitHub
3. Import GitHub repo
4. 自動部署，得到網址

## 方法三：Netlify Drop

1. 前往 https://app.netlify.com/drop
2. 將 `out` 資料夾拖進去
3. 立即得到網址
